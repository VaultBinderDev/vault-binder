Pokemon Collector Page

This is my first Pokemon project.

This is a fan made Pokemon site, designed to give card collectors a fan base created platform to store a visual card for their own personal collection. You can add cards manually, and soon automatically using basic imaging software. You can favorite cards, search by multiple details such as HP, Type, and name, alpahbetically, or via the search bar for duplicates of the same name.
