# Vault Binder Dev LOg

## Beta Build --- First Public Version

### Project Goal
Vault Binder started as a personal Collection Space for my own Pokemon Cards, but it then turned into a Learning and Collection tool that others could use. 

The goal for this site is to help others understand Pokemon Cards better, and to give the ones who do collect a simple space to hold their cards, free of charge, and to organize and sort their collections easier. 
______

### Major Features Built-In
- Created a landing page for searching Pokemon Cards
- Connected the Pokemon TCG API
- Added secure API calls to Netlify
